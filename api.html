CyberEx API Docs

CyberEx API, yüksek frekanslı ticaret ve güvenli varlık yönetimi için tasarlanmış, düşük gecikmeli bir REST ve WebSocket arayüzü sunar.

Kimlik Doğrulama

Authentication

Tüm özel uç noktalar (private endpoints), API Key ve Secret Key ile imzalanmış istekler gerektirir. HMAC SHA256 kullanıyoruz.

All private endpoints require requests signed with API Key and Secret Key. We use HMAC SHA256.

Example Request (Ticker)
