  GITHUB SECRETS KURULUMU - OTOMATIK DEPLOYMENT
═══════════════════════════════════════════════════════════

Ben otomatik deployment sistemini hazırladım!
Şimdi sadece GitHub'da secrets eklemen gerekiyor.

═══════════════════════════════════════════════════════════
ADIM 1: GitHub Repository'ye Git
═══════════════════════════════════════════════════════════

https://github.com/CyberEx/web3app
(veya kendi repository URL'in)

═══════════════════════════════════════════════════════════
ADIM 2: Settings > Secrets
═══════════════════════════════════════════════════════════

1. Repository sayfasında üstteki "Settings" sekmesine tıkla
2. Sol menüden "Secrets and variables" > "Actions" seç
3. "New repository secret" butonuna tıkla

═══════════════════════════════════════════════════════════
ADIM 3: Secrets Ekle
═══════════════════════════════════════════════════════════

SECRET 1:
  Name: FTP_USERNAME
  Value: (FTP kullanıcı adın)
  → "Add secret" tıkla

SECRET 2:
  Name: FTP_PASSWORD
  Value: (FTP şifren)
  → "Add secret" tıkla

═══════════════════════════════════════════════════════════
ADIM 4: Otomatik Deployment Aktif!
═══════════════════════════════════════════════════════════

Artık her git push yaptığında:
  ✅ GitHub Actions otomatik çalışacak
  ✅ web/ klasöründeki dosyalar otomatik yüklenecek
  ✅ cyberex.com.tr/web3app otomatik güncellenecek!

═══════════════════════════════════════════════════════════
MANUEL TETİKLEME
═══════════════════════════════════════════════════════════

GitHub'da:
  1. "Actions" sekmesine git
  2. "Auto Deploy to cyberex.com.tr/web3app" workflow'unu seç
  3. "Run workflow" butonuna tıkla
  4. Branch seç: master veya main
  5. "Run workflow" tıkla

═══════════════════════════════════════════════════════════
NOT: Eğer web3wallet veya web3extension'da secrets varsa
═══════════════════════════════════════════════════════════

Aynı FTP bilgilerini kullanabilirsin!
Sadece secrets'ları ekle, workflow otomatik çalışacak.
