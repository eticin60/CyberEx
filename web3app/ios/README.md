# CyberEx Web3 Wallet - iOS

iOS uygulaması için SwiftUI tabanlı proje yapısı.

## Gereksinimler

- Xcode 15.0+
- iOS 14.0+
- Swift 5.9+

## Kurulum

1. Xcode'da projeyi açın
2. Pod install çalıştırın (CocoaPods kullanılıyorsa)
3. Build ve Run

## Özellikler

- SwiftUI ile modern arayüz
- Web3 entegrasyonu
- Biyometrik kimlik doğrulama
- Token yönetimi
- Çoklu ağ desteği

